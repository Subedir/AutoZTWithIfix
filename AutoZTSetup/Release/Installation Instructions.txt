ZenoTrac Automation Tool Installation


1.  If AutoZenoTrac has already been installed, then uninstall the old AutoZenoTrac version from Add or Remove Programs
2.  Download Setup.exe and AutoZenoTrac.msi into the same folder and double click on setup.exe to install.
3.  Run AutoZT.exe (Start - All Programs - AutoZT) , if not successful go to step 3; otherwise you are done.
4.  Download Interop Assemblies folder and double click to run dO2003PIA.MSI .
5.  Run AutoZT.exe again. Done.
